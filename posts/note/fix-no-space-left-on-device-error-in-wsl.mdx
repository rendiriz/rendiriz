---
type: Note
title: Fix "No space left on device" Error in WSL
date: 2024-02-03
---

```bash
wsl.exe --shutdown
```

```bash
(Get-ChildItem -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Lxss | Where-Object { $_.GetValue("DistributionName") -eq '<distribution-name>' }).GetValue("BasePath") + "\ext4.vhdx"
```
replace `<distribution-name>` with the actual distribution name.\
Example: Ubuntu-20.04

```bash
diskpart
```

```bash
Select vdisk file="<pathToVHD>"
```
replace `<pathToVHD>` with the actual path to the VHD.\
Example: C:\Users\User\AppData\Local\Packages\CanonicalGroupLimited.UbuntuonWindows_79rhkp1fndgsc\LocalState\ext4.vhdx

```bash
detail vdisk
```

```bash
expand vdisk maximum=<sizeInMegaBytes>
```
replace `<sizeInMegaBytes>` with the actual size in MegaBytes.\
Example: 512000

```bash
exit
```
